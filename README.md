# Loan Risk Analysis & Prediction using Python

## Project Overview
This project analyzes loan applicant data to identify factors influencing loan approval and builds a machine learning model to predict loan approval status.
The objective is to simulate a real-world NBFC/FinTech risk analytics use case using Python.

---

## Business Problem
Financial institutions need to assess applicant risk before approving loans. Incorrect approvals can increase defaults and financial losses.

This project helps identify:
- Key approval drivers
- Risk-related applicant patterns
- Predictive indicators for loan approval

## Key Insights
- Applicants with good credit history showed significantly higher approval rates.
- Loan amount and applicant income influenced approval probability.
- Semi-urban applicants showed relatively higher approval trends in this dataset.
- Credit history emerged as the strongest predictive feature.
